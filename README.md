# shoppingMall
购物商城管理系统带万字文档 在线商城系统Java vue SpringBoot 前后端分离 web网站

语言:Java 数据库:MySQL

前端技术:vue、element-ui

后端技术: springboot、mybatisplus、redis

系统角色:管理员、用户

管理员:登录、商品管理、分类管理、广告管理、头像管理、文件信息管理、图表分析、收入排行等

用户:注册、登录、商城首页、购物车信息、商品分类信息、订单信息
